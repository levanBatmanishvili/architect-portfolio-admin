# Architect Portfolio Admin

## Project Overview
This project is an architect portfolio website with an admin interface.

It was built to practice:
- DOM manipulation
- API communication
- login functionality
- dynamic gallery management
- front-end structure

---

## Main Features
- Portfolio gallery display
- Dynamic project filtering
- Admin login page
- Add and delete portfolio items
- Modal management
- API integration

---

## Technologies Used
- HTML
- CSS
- JavaScript
- REST API

---
